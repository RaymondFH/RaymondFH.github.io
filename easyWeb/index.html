How to Use the Webpage Builder

  1. Enter HTML: Start by typing or pasting your HTML content into the "HTML" text area. Use Markdown-style headings (e.g., # for H1, ## for H2) and blank lines for paragraphs.
  2. Customize Styles: Use the various options in the "CSS Customizations" section to adjust the appearance of your webpage:
    • Select a font family from the dropdown menu
    • Adjust font size, line height, and letter spacing using the sliders
    • Choose text alignment and font weight
    • Set border, margin, and padding values
  3. Set Colors: Use the color pickers in the "Customizations" section to set the background and text colors.
  4. Add Effects: Toggle the checkboxes to add box shadow or text shadow effects.
  5. JavaScript Options: Enable click alerts or hover effects using the checkboxes in the "JavaScript Options" section.
  6. Use Templates: Click on the template buttons to quickly populate the HTML area with pre-designed content.
  7. Preview: Your changes will be reflected in real-time in the preview pane on the right (or below on mobile devices).
  8. Download: Once you're satisfied with your webpage, use the "Download HTML" and "Download CSS" buttons to save your creation.

Tip: Experiment with different combinations of styles and options to create your perfect webpage!
